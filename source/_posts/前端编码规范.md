### 关于css注释

在查看人家的网页中，我发现了如下的注释方式：
```css
/* ========
  基本样式
  ========*/
...

/* ========
  头部样式
  ========*/
...
```

```css
/* 基本样式 */
...
/* /基本样式 */
```
我个人感觉第一种比较好，所以我决定在项目中采用第一种。
